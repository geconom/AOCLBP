Καλησπέρα σας,

Σας στέλνω όλες τις υλοποιήσεις του backprojection ώστε να υπάρχουν μαζεμένες.

Direct Μέθοδοι

    direct_par_fgpa => H direct μέθοδος που εκτελείται παράλληλα με NxN workitems χωρίς να έχω χωρίσει τα items σε γκρουπ (./program N N input.txt)
    direct_par_tiles => Η direct μέθοδος που δίνουμε ένα divide_num σαν είσοδο και μας τρέχει τον αλγόριθμο παράλληλα σε tiles μεγέθους divide_num^2 αντί να το κάνει παράλληλα για κάθε pixel (./program N N input.txt divide_num)
    direct_par_workgroups => Η direct μέθοδος που εκτελείται παράλληλα για NxN workitems χωρισμένα σε γκρουπ των οποίων το μέγεθος workgroup size δίνεται από την είσοδο (./program N N input.txt workgroup_size). Η διαφορά σε σχέση με το παραπάνω είναι ότι όλα γίνονται παράλληλα μεταξύ των tiles και μεταξύ των στοιχείων των tiles. Αντίθετα στον direct_par_tiles κώδικα οι υπολογισμοί στο κάθε tile γίνονται σειριακά.

Τα παραπάνω τα συγκρίνω με τον σειριακό κώδικα serial_direct (./program N N input.txt)

Μέθοδοι με αναδρομή

    direct_par_wd_1 => Ο αλγόριθμος τρέχει αναδρομικά μέχρι να φτάσω σε ένα base size (χρησιμοποιώ 128). Τότε εκτελείται παράλληλα ο αλγόριθμος direct για καθεμια από τις υποεικόνες. (./program N N basesize input.txt)

Τα παραπάνω τα συγκρίνω με τον σειριακό κώδικα serial_bp (./program N N basesize input.txt)

Για όλα τα παραπάνω δοκιμάζω διαφορετικές interpolation functions στους kernels.

Για οτιδήποτε χρειαστείτε πείτε μου.
